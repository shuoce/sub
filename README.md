# 📖 使用教程

## NekoBox

1. 打开 NekoBox
2. 进入「订阅管理」
3. 点击添加订阅
4. 粘贴：

```

https://shuoce1.eu.cc/sub.txt
```

5. 更新订阅即可


---

## Clash / Mihomo

1. 打开 Clash 客户端
2. 找到「配置 / Profiles」
3. 添加远程配置
4. 粘贴：

```

https://shuoce1.eu.cc/share/synchronous
```

5. 下载并启用配置
